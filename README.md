# simple-ts-app
  
## npm scripts の説明

このプロジェクトで使える npm scripts の内容と使い方は以下の通りです。

### dev
- **コマンド:** `npm run dev`
- **説明:** 開発用サーバーを起動します。ファイルを編集すると自動でリロードされ、ブラウザで動作確認ができます。

### build
- **コマンド:** `npm run build`
- **説明:** TypeScript のコンパイル（`tsc`）を行い、その後 Vite で本番用のビルド（`vite build`）を作成します。本番公開用の最適化されたファイルが生成されます。

### preview
- **コマンド:** `npm run preview`
- **説明:** `build` で作成した本番用ファイルをローカルサーバーで確認できます。本番環境に近い動作をテストできます。

### test
- **コマンド:** `npm test` または `npm run test`
- **説明:** テストを実行します。`vitest` というテストツールで、プロジェクト内のテストコードが動きます。
